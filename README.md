# CursoMLGoogle

Repositorio con ejercicios del curso de Machine Learning de Google.

## Descripción

Este repositorio recopila ejercicios prácticos para aprender y practicar conceptos de Machine Learning. La mayoría de los ejercicios incluidos son ejemplos ya hechos y publicados por Google en sus materiales educativos (principalmente en Google Colab), y algunos han sido adaptados por el autor para facilitar el aprendizaje y la experimentación.

## Contenido

- Notebooks con ejercicios y ejemplos prácticos.
- Ejercicios de Google (mayoría) y algunos propios/adaptados.
- Material listo para usar en Google Colab o localmente con Jupyter.

## Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/JesusLuna2309/CursoMLGoogle.git
   ```

2. Abre los notebooks en Google Colab o localmente si tienes instalado Jupyter.
